# Cervantes User Guide

## Inicio

1. Ejecuta `npm run db:push`.
2. Ejecuta `npm run dev`.
3. Abre `http://localhost:5173`.

## Flujo de trabajo

1. Crea un proyecto desde el dashboard.
2. Genera preguntas en Idea Intake y responde lo relevante.
3. Avanza por investigación, idiomas, GO/NO-GO, fórmula y biblias.
4. Genera el índice maestro.
5. Crea bloques de manuscrito, edítalos y apruébalos.
6. Ejecuta auditoría, recuperación y exportaciones.
7. Genera metadata y checklist de publicación.

## Modo IA

Sin `OPENAI_API_KEY`, Cervantes usa plantillas editables. Con una API key configurada, `AIService` queda listo para conectar generación real.
